# Sleep Habit Tracking System

A web-based application to help users record, manage, and analyze their daily sleep habits.

## 🛠️ Built With

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

## ✨ Features

- User registration and secure login
- Add, edit, and delete sleep records
- Automatic sleep duration calculation
- View complete sleep history
- Sleep quality tracking (Excellent / Good / Fair / Poor)
- Session-based authentication
